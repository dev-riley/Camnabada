## 캠나바다(CAM:NABADA)

#### 캠핑 나누고 바꾸고 다시쓰자!!

- 캠핑을 하러가면 불필요하게 음식을 많이 사거나 챙기지 못한 물건들이 있기 마련
- 캠핑을 처음하는 사람들을 위한 캠핑 관련 정보를 제대로 제공하는 플랫폼 부재
- 캠핑을 즐기는 사람들의 커뮤니티 필요성
